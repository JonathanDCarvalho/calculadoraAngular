# Calculadora em Angular

Este é um projeto de uma calculadora simples desenvolvida em Angular, utilizando TypeScript e Sass (SCSS) para estilização. A calculadora permite operações matemáticas básicas e usa a biblioteca mathjs para realizar cálculos de forma segura.

Este projeto foi gerado com [Angular CLI](https://github.com/angular/angular-cli) versão 18.2.11.

## Tecnologias Utilizadas

- *Framework*: Angular
- *Linguagens*: TypeScript, HTML e SCSS (Sass)
- *Biblioteca para Cálculo*: [mathjs](https://mathjs.org/)

## Funcionalidades

- Realiza operações matemáticas básicas: adição, subtração, multiplicação e divisão.
- Exibe os operadores e os números na tela de forma clara.
- Botão de "C" para limpar a tela e botão de "=" para calcular o resultado.

## Pré-requisitos

Antes de iniciar, é necessário ter o seguinte instalado em seu ambiente:

- [Node.js](https://nodejs.org/) (recomendado: versão 14 ou superior)
- [Angular CLI](https://angular.io/cli) (recomendado: versão 12 ou superior)
